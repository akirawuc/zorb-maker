# zorb-maker

Zorb-maker enables you to customize your own zorb using the color scheme you provide.

## Usage

- Choose and upload the image you want to retrieve the color scheme from.

- The app will calculate the dominant color of your image using k-means, and get the top-5 colors.

- A Zorb will appear base on the calculated result (in a specific order).

- You can switch the order of the color base on your preference.
